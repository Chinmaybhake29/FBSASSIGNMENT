Core Java

Javais a high-level, object-oriented programming language. It's easy to learn and widely used for building: 
        Web apps

Mobile apps (especially Android)

Desktop software

Games

Big enterprise systems

Main Features:
Simple – Easy to read and write.

Platform-independent – "Write once, run anywhere" (thanks to Java Virtual Machine – JVM).

Object-oriented – Code is organized using objects and classes.

Secure – Has strong security features.

Multithreaded – Can run many tasks at the same time.
